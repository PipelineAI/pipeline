Key Parameters

PIO_MODEL_SERVER_HOST Example: predict.community.pipeline.io
PIO_MODEL_TYPE Available Model Types: tensorflow, scikit, spark, xgboost, r, java, python3
PIO_MODEL_NAME Examples: census-2017, airbnb-san-francisco, etc.

Deploy


            curl -i -X POST -H "Transfer-Encoding: chunked" \
              -F "file=@pipeline.tar.gz" \
              http://$PIO_MODEL_SERVER_HOST:6969/api/v1/model/deploy/$PIO_MODEL_TYPE/$PIO_MODEL_NAME \
              -w "\n\n"

Predict


            curl -X POST -H "Content-Type: application/json" \
              -d '{"x_observed":1.5}' \
              http://$PIO_MODEL_SERVER_HOST:6969/api/v1/model/predict/$PIO_MODEL_TYPE/$PIO_MODEL_NAME \
              -w "\n\n"

Upload New Model Version


               pio init-model --model-server-url={model_server_url} \
               --model-type={model_type} \
               --model-namespace={model_namespace} \
               --model-name={model_name} \
               --model-version={model_version} \
               --model-path /path/to/model/directory/ \
               --model-test-request-path /path/to/test_request.json
              

               pio deploy

Predict


              pio predict
